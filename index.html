# Bill Discounting API Documentation

## Buyer Limit Creation

URL: `POST /limit`

Request:
```sql
select * from table;
import java.nio.8;
{
	"data": {
		"partyType": "BUYER",
		"customerId": 248427,
		"customerLimit": "1000000",
		"odsanctionLimit": "1000000",
		"startDate": "2022-04-01",
		"limitValidity": "1",
		"endDate": "2023-04-01",
		"disbursementValidity": "2023-04-01",
		"revolving": true
	},
	"action": "SAVE", // Static value
	"processType": "Customer", // Static value
	"processName": "Limit", // Static value
	"currentStage": "Complete" // Static value
}
```
